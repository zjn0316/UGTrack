#### 训练（通用入口）

```python
# OSTrack
python tracking/train.py --script ostrack --config vitb_256_mae_32x4_ep300 --save_dir ./output --mode single
# UGTrack
python tracking/train.py --script ugtrack --config s1_best_t10_bce05 --save_dir ./output --mode single
```

#### 训练（UWB 专用入口）

```python
python tracking/train_uwb.py --script ugtrack --config s1_best_t10_bce05 --save_dir output
```

#### 测试（通用入口）

```shell
# OSTrack
python tracking/test.py ostrack vitb_256_mae_ce_32x4_ep300 --dataset_name otb100_uwb

python tracking/test.py ugtrack s1_best_t10_bce05 --dataset_name otb100_uwb
```
